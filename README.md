# robotis_lab

This repository provides tutorials and tools for reinforcement learning (RL) and imitation learning using ROBOTIS robots. It enables training policies in simulation and deploying them on real hardware via Sim2Real functionality, facilitating research and development in robot learning.

## Key Features
*   Tutorials for reinforcement and imitation learning with simulated ROBOTIS robots.
*   Sim2Real pipeline for transferring trained policies to physical robots.
*   Support for various learning environments and robot models.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   OpenAI Gym / Gazebo
*   PyTorch / TensorFlow

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/robotis_lab.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the setup instructions in the `docs/` directory for your specific robot and simulation environment.
